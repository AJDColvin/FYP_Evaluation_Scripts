# Evaluation Scripts

This repository contains the evaluation scripts and test media used to calculate performance metrics for the [Horse Tracker](https://github.com/AJDColvin/Horse_Tracker) project.

## Evaluated Metrics

* **Vision-Language Models (VLMs):** Peak $F_1$ scores
* **Real-Time Object Detection Models:** Peak $F_1$ scores
* **Multi-Object Tracking (MOT) Algorithm:** Tracking accuracy

For the primary implementation and core codebase, please visit the main project repository: [AJDColvin/Horse_Tracker](https://github.com/AJDColvin/Horse_Tracker).
